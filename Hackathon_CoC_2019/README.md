# ICT CoC Hackathon

하수구 이미지에서 결함을 탐지하기 위한 분류 코드. 
Backbone 모델로부터 분류기를 지식 전이 학습함.

## Data
- 2D images from video
- Data is not publicly available.

## Model Architecture
- Backbone models (e.g.vgg, resnet...) + classifier

## Requirements
```
pytorch 1.0.0+\
python 3.6+ \
```

## Usage
```bash

python finetuning_classifier.py

```
